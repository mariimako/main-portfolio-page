Recall App

A full-stack group project for nwHacks 2023. A mobile app created with React Native as front end aimed to assist dementia patients with associating names with faces by creating an OpenCV image similarity program, an SQL database to store faces and names, and using flask to connect the back and front end. Working with three other members, I developed the OpenCV image similarity program.
